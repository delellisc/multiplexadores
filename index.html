MULTIPLEXADORES

O QUE É UM MULTIPLEXADOR?

Os multiplexadores são dispositivos que permitem a seleção de uma de várias entradas de dados para direcioná-las a uma única saída, sendo amplamente utilizados em sistemas digitais (Tocci, Widmer e Moss, 2011).

MULTIPLEXADOR DE DUAS ENTRADAS

Um multiplexador de duas entradas é um circuito lógico que possui duas entradas de dados, uma entrada de seleção e uma saída. O nível lógico aplicado na entrada de seleção determina qual das portas AND será ativada, permitindo que o dado correspondente passe pela porta OR e chegue à saída. A expressão booleana que representa esse comportamento é X = E₀·S̅ + E₁·S, onde S̅ é o complemento da entrada de seleção S.

MULTIPLEXADOR DE QUATRO ENTRADAS

Um multiplexador de quatro entradas possui quatro entradas de dados (E0, E1, E2, E3) e duas entradas de seleção (S1 e S0), que determinam qual das entradas de dados será transmitida para a saída (X). A seleção é feita de acordo com as combinações possíveis dos valores nas entradas de seleção. A expressão lógica para o multiplexador de quatro entradas é dada por: X = (E0 · S1 · S0) + (E1 · S1 · S0) + (E2 · S1 · S0) + (E3 · S1 · S0).
Isso significa que a entrada E0 será selecionada quando S1 = 0 e S0 = 0, a entrada E1 será selecionada quando S1 = 0 e S0 = 1, a entrada E2 será selecionada quando S1 = 1 e S0 = 0, e a entrada E3 será selecionada quando S1 = 1 e S0 = 1. Assim, dependendo dos valores de S1 e S0, apenas uma das entradas de dados será transmitida para a saída.

MULTIPLEXADOR DE OITO ENTRADAS

Um multiplexador de 8 entradas possui 8 entradas de dados (E0, E1, E2, E3, E4, E5, E6, E7) e três entradas de seleção (S2, S1 e S0), responsáveis por determinar qual entrada de dados será encaminhada para a saída (X). A seleção é realizada com base nas diferentes combinações possíveis dos valores nas entradas de seleção. A expressão lógica para o multiplexador de 8 entradas é dada por: X = (E0 · S2 · S1 · S0) + (E1 · S2 · S1 · S0) + (E2 · S2 · S1 · S0) + (E3 · S2 · S1 · S0) + (E4 · S2 · S1 · S0) + (E5 · S2 · S1 · S0) + (E6 · S2 · S1 · S0) + (E7 · S2 · S1 · S0).
Ou seja, a entrada E0 será selecionada quando S2 = 0, S1 = 0 e S0 = 0, a entrada E1 será escolhida quando S2 = 0, S1 = 0 e S0 = 1, a entrada E2 será selecionada quando S2 = 0, S1 = 1 e S0 = 0, a entrada E3 será escolhida quando S2 = 0, S1 = 1 e S0 = 1, a entrada E4 será ativada quando S2 = 1, S1 = 0 e S0 = 0, a entrada E5 será selecionada quando S2 = 1, S1 = 0 e S0 = 1, a entrada E6 será transmitida quando S2 = 1, S1 = 1 e S0 = 0, e a entrada E7 será escolhida quando S2 = 1, S1 = 1 e S0 = 1. Dessa forma, dependendo do valor das entradas de seleção, apenas uma das entradas de dados será enviada para a saída.
